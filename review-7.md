## 残存する問題

### [小] ステータス表示のキャパシタ情報が二重・不統一 (前回から継続)

修正によって L184 に「キャパシタ残量」が追加されましたが、L188 の「ジェネレータのキャパシタ容量」が残ったため、同じステータス欄に類似情報が2行表示される状態になっています。

```
L184: 燃料残量、キャパシタ残量、エネルギー発生率 (gj/sec)
L188: ジェネレータのキャパシタ容量          ← 残量か最大容量か依然不明確
```

L188 を「キャパシタ最大容量 (gj)」と明示したうえで「キャパシタ残量 / キャパシタ最大容量」のように1行にまとめるか、L188 を削除することを推奨します。

### [小] ジェネレータの「キャパシタ現在量」と「現在容量」が重複

L410 と L412 に同じ内容と思われる属性が2つあります:

```
L410: キャパシタ現在量 (gj)
L412: 現在容量 (gj)
```

「現在容量」は「キャパシタ現在量」の別名と見られます。どちらか一方を削除してください。

---

### [小] ジェネレータの振る舞い名「発生率設定」が属性名と不一致

L411 で属性名が「発生率」から「エネルギー発生率」に変更されましたが、L422 の振る舞い名はそのまま残っています:

```
L411: エネルギー発生率 (gj/sec)   ← 属性名（更新済み）
L422: 発生率設定                  ← 振る舞い名（未更新）
```

「エネルギー発生率設定」に揃えてください。
